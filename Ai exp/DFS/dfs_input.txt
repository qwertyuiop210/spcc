Depth First Search (DFS)

Definition:
DFS is a graph traversal technique that explores a node, then goes as deep as possible along each branch before backtracking.

How it works:

Start from a node
Visit it
Recursively visit its first unvisited neighbour
Continue until no more nodes → backtrack

Data Structure Used: Stack (via recursion)

Applications:

Path finding
Cycle detection
Topological sorting
✅ Example Graph
        A
       / \
      B   C
     / \
    D   E

Adjacency List:

A → B, C
B → D, E
C → -
D → -
E → -

DFS Traversal (starting from A):

A → B → D → E → C

Differences

| Feature        | DFS          | BFS                   |
| -------------- | ------------ | --------------------- |
| Strategy       | Depth-wise   | Level-wise            |
| Data Structure | Stack        | Queue                 |
| Path           | Not shortest | Shortest (unweighted) |
| Memory         | Less         | More                  |
