Breadth First Search (BFS)

Definition:
BFS is a graph traversal technique that explores nodes level by level, visiting all neighbours before going deeper.

How it works:

Start from a node
Visit all its neighbours first
Then move to next level

Data Structure Used: Queue

Applications:

Shortest path in unweighted graphs
Level-order traversal
Network broadcasting



        A
       / \
      B   C
     / \
    D   E


    A → B → C → D → E



Differences

| Feature        | DFS          | BFS                   |
| -------------- | ------------ | --------------------- |
| Strategy       | Depth-wise   | Level-wise            |
| Data Structure | Stack        | Queue                 |
| Path           | Not shortest | Shortest (unweighted) |
| Memory         | Less         | More                  |
