Definition

Uniform Cost Search (UCS) is a search algorithm that expands the node with the lowest path cost from the start node.

Working
Start from source node
Always select node with minimum cost
Update cost of neighbors
Continue until goal is reached
Characteristics
Uses priority queue (or min-cost selection)
Similar to Dijkstra’s Algorithm
Works for weighted graphs
Advantages
Finds optimal (least cost) path
Complete algorithm
Works with varying edge costs
Disadvantages
Slower than BFS
Uses more memory
Explores many nodes
