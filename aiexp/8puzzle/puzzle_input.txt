Definition

The 8-puzzle problem consists of a 3×3 grid with 8 numbered tiles and one blank space.
The objective is to reach the goal state by sliding tiles into the blank space.

A* in 8-Puzzle

A* uses:

g(n) = number of moves so far
h(n) = number of misplaced tiles
f(n) = g(n) + h(n)

It always selects the state with the lowest f(n).

Advantages
Finds optimal solution
Uses heuristic to reduce search space
Efficient compared to brute-force
Disadvantages
High memory usage
Time complexity increases for difficult states
